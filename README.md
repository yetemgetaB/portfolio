# Portfolio Website

A modern, interactive personal portfolio website showcasing my projects, skills, and journey as a Computer Science student at Unity University.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Dark/Light Theme Toggle** - Switch between themes with smooth transitions
- **Particle Background Animation** - Interactive canvas animation on hero section
- **Animated Typewriter** - Cycling text showcasing multiple roles
- **Scroll Progress Indicator** - Visual progress bar at the top
- **Animated Counters** - Numbers animate when scrolling into view
- **3D Tilt Effect** - Cards tilt on mouse hover
- **Skill Progress Bars** - Animated progress bars for each skill
- **GitHub Stats Integration** - Live repository and star counts
- **Working Contact Form** - Formspree integration for email submissions
- **Smooth Scroll Navigation** - Keyboard (arrow keys) and click navigation
- **Toast Notifications** - Success/error feedback messages

## Tech Stack

- **HTML5** - Semantic markup with SEO meta tags
- **CSS3** - Custom properties, animations, grid/flexbox layouts
- **JavaScript** - Vanilla JS with Intersection Observer API
- **Canvas API** - Particle animation system
- **Formspree** - Contact form backend
- **Font Awesome** - Icons
- **Devicon** - Technology icons

## Pages

- **Home** - Hero section with animated intro
- **About** - Bio with profile photo
- **Skills** - Technical skills with progress bars
- **GitHub Stats** - Live contribution graph and stats
- **Projects** - Featured project showcase
- **Resume** - Education timeline with downloadable CV
- **Contact** - Contact form and social links
